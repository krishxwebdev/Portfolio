# Resume Placeholder

Place your resume PDF in this folder (`public/resume/`) and name it `Krishna_Resume.pdf`.

The "Download Resume" button in the Hero section links to `/resume/Krishna_Resume.pdf`.

Since Vite serves files from the `public/` directory at the root URL, 
placing it at `public/resume/Krishna_Resume.pdf` will make it available at `/resume/Krishna_Resume.pdf`.

Example:
- File location: `client/public/resume/Krishna_Resume.pdf`
- URL in production: `https://yoursite.com/resume/Krishna_Resume.pdf`
